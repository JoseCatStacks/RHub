# RHub • Launchpad

The next-generation crypto launchpad and hub built on Robinhood Blockchain.

## Features

- **Launchpad** — Discover and invest in new token launches
- **Real Token Deployment** — $RHUB genesis token live on Robinhood Blockchain
- **Staking** — Earn yield on $RHUB and other assets
- **Trading** — Fast, low-fee trading interface
- **Portfolio** — Track all your holdings in one place

## Tech Stack

- Single-file HTML + Tailwind CSS
- Vanilla JavaScript
- Chart.js for trading charts
- Fully responsive and interactive

## Local Development

Simply open `index.html` in any browser. No build step required.

## Deployment

This project is deployed on **Vercel** at [rhub.fun](https://rhub.fun)
